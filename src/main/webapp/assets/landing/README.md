# Landing Page Assets

## 视频资源（必填，影响首屏效果）

当前 `video/hero.mp4` 尚未放置真实视频文件。

**推荐下载（免费可商用）：**

1. Pexels: https://www.pexels.com/search/videos/design%20art%20creative/
   - 搜索 "abstract design" 或 "creative poster" 或 "art digital"
   - 下载 1080p MP4，重命名为 `hero.mp4`，放入此目录

2. Coverr: https://coverr.co/
   - 搜索 "abstract" 或 "motion"
   - 下载 MP4，体积控制在 15MB 以内

3. Mixkit: https://mixkit.co/free-stock-video/
   - 搜索 "art" / "design" / "light"

**视频要求：**
- 格式：MP4 (H.264)
- 时长：5–15 秒，可循环
- 分辨率：1280×720 或 1920×1080
- 体积：控制在 15MB 以内（源码压缩包限制）

## 海报图片资源（当前为 SVG 占位符）

当前 `images/` 目录下的 poster-0N.jpg 为 SVG 格式占位图，
可在时间允许时替换为真实海报设计图或校园设计作品图片。

**如需替换：**
- 尺寸：3:4 比例，建议 400×533px 以上
- 格式：JPG 或 WebP
- 体积：每张 200–500KB
- 内容：设计类海报、校园竞赛作品、创意展览图片

**推荐免费素材：**
- Unsplash: https://unsplash.com/s/photos/poster-design
- Pinterest（截图参考）
- 使用团队自己的竞赛作品（最佳选择）
